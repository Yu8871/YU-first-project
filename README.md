# YU-first-project

我的第一个 Python 练习项目 🐍

## 项目说明

这个项目包含三个初级 Python 练习项目，适合新手学习 Python 编程基础。

## 项目列表

### 1. 简易计算器 (`calculator.py`)

一个基础的四则运算计算器。

**学习要点：**
- ✅ 函数的定义和调用
- ✅ 异常处理（try-except）
- ✅ 循环控制（while）
- ✅ 用户输入和输出
- ✅ 条件判断（if-elif-else）

**功能：**
- 支持加、减、乘、除四种运算
- 完整的错误处理（如除以零）
- 菜单式交互界面
- 循环操作直到用户选择退出

**运行方式：**
```bash
python calculator.py
```

---

### 2. 猜数字游戏 (`guessing_game.py`)

一个有趣的交互式猜数字游戏，包含多个难度等级。

**学习要点：**
- ✅ 随机数生成（random 模块）
- ✅ 循环控制（while）
- ✅ 条件判断（if-elif-else）
- ✅ 用户输入验证
- ✅ 游戏计数和评分

**功能：**
- 3 个难度等级（简单、中等、困难）
- 智能提示（太大/太小）
- 剩余机会提示
- 性能评价（根据尝试次数）
- 无限重玩

**难度设置：**
| 难度 | 范围 | 机会 |
|------|------|------|
| 简单 | 1-10 | 7次 |
| 中等 | 1-50 | 8次 |
| 困难 | 1-100 | 10次 |

**运行方式：**
```bash
python guessing_game.py
```

### 3. MBTI 性格测试 (`mbti_test.py`) ⭐ 新增

一个交互式的性格测试问卷，帮助用户了解自己的 MBTI 类型。

**学习要点：**
- ✅ 字典数据结构（数据组织）
- ✅ 列表遍历和循环
- ✅ 字符串处理和格式化
- ✅ 函数的模块化设计
- ✅ 数据统计和分析
- ✅ 条件判断逻辑

**功能：**
- 8 道精心设计的题目
- 自动计算 MBTI 16 种类型
- 详细的性格描述和分析
- 维度分析展示
- 支持重复测试

**测试原理：**
MBTI 由 4 个维度组成，每个维度有两个对立的特征：
- **E/I** - 外向 vs 内向（能量来源）
- **S/N** - 感知 vs 直觉（信息收集方式）
- **T/F** - 思维 vs 情感（决策方式）
- **J/P** - 判断 vs 知觉（生活方式）

**16 种性格类型：**
按不同维度组合产生 16 种独特的性格类型，每种都有独特的名称和特点。

**运行方��：**
```bash
python mbti_test.py
```
